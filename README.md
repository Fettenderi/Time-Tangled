# Time Tangled

is a Bullet Hell space game made during the [Bullet Hell Jam](https://itch.io/jam/bullet-jam-2021).

# Controls

WASD or Left Stick to Move

Arrows or Action Buttons to Shoot

R  or L1 to Restart
